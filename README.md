# 🎉 BINGO LIVE - Real-time Sync System

ブラウザだけで完結する、リアルタイム同期型のビンゴ抽選システムです。
専用のサーバーを用意することなく、主催者の操作を参加者の画面へ即座に反映します。

## ✨ 特徴

- **リアルタイム同期**: PeerJSを利用したP2P通信により、抽選結果をリアルタイムに共有。
- **URL招待機能**: 専用のURLを送るだけで、参加者はID入力なしで自動接続。
- **状態保持（リロード対策）**: 主催者が誤ってブラウザを閉じても、履歴を保持したまま再開可能（LocalStorage利用）。
- **演出重視**: ワクワク感を煽るシャッフルエフェクト付き。

## 🚀 使い方

### 1. 運営者（ホスト）
1. `https://tamachamo.github.io/bingo/` にアクセスします。
2. 「招待URLをコピー」ボタンを押し、参加者にLINEなどで送ります。
3. 全員の接続を確認したら「数字を引く！」ボタンで抽選を開始します。

### 2. 参加者（ビューアー）
1. 送られてきたURL（末尾に `?id=...` が付いたもの）を開きます。
2. 「同期中 ✅」と表示されれば準備完了です。あとは運営者の操作を待つだけ！

## 🛠 テクノロジー

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Communication**: [PeerJS](https://peerjs.com/) (WebRTC)
- **Deployment**: GitHub Pages

## ⚠️ 注意事項

- 主催者がページを再読み込みすると通信IDが変わるため、URLの再共有が必要になります（抽選履歴は保持されます）。
- 安定した通信環境（Wi-Fi推奨）でご利用ください。
